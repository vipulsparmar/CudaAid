Features

Everything you need to debug your stack

From GPU hardware detection to PyTorch version matching — CudaAid covers the entire diagnostics pipeline.

🖥️

GPU Detection

Reads device name, driver version, VRAM, and compute capability directly from nvidia-smi with structured query fallbacks.

⚡

CUDA Detection

Checks toolkit version via nvcc and runtime version via torch.version.cuda. Handles missing installations gracefully.

🔥

PyTorch Detection

Reads version, CUDA build tag, cuDNN version, and torch.cuda.is_available() with detailed status reporting.

🔗

Compatibility Engine

Built-in driver → CUDA → PyTorch matrix. Automatically recommends the best stable version combo for your hardware.

🩹

Auto-Fix

Generates pip install commands to resolve mismatches. Supports dry-run mode to preview before executing.

📤

JSON Export

Export full diagnostics as structured JSON with metadata — perfect for CI pipelines, logs, and team sharing.

CLI Commands

Four commands, zero friction

Each command runs the full detection pipeline and presents results in the format you need.

cudaaid check
Full environment diagnostic with GPU, CUDA, and PyTorch status badges and compatibility verdict.
cudaaid explain
Human-friendly paragraph explanation of your current setup — great for sharing with teammates.
cudaaid fix --dry-run
Show the pip commands that would fix compatibility issues, without actually running them.
cudaaid export -f json
Export full diagnostics as structured JSON with metadata — ideal for CI/CD pipelines.
